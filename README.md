# study-planner
A simple study planner with the ability to log study sessions.

This small project was a way for me to begin practicing/using Agentic AI, specifically Codex. None of the raw files were edited by me which forced me to test how far I could push AI to achieve my goal for this project. It's been a super cool experience integrating it with my IDE, and I'm excited to keep testing the waters! 
Students can use this planner to log courses, course grades, and assignments for each course. Assignments are automatically updated and seen in a calendar view that can be toggled between a monthly-view and a weekly-view. Users may also log "study" sessions, which are then summarized daily and stored to look back on.
